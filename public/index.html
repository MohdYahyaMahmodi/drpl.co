About FileDrop

FileDrop is a simple, secure way to share files between devices on the same network.

Key Features:

  • No registration or internet required
  • Peer-to-peer encrypted transfers
  • Works on any device with a web browser
  • No file size limits
  • Files are never stored on servers

How it Works:

  • Open FileDrop on both devices
  • Select the receiving device
  • Choose files to send
  • Files are transferred directly and securely
